# LearnHub Online Courses API

## Description
This is a FastAPI backend project for an online course platform where users can browse courses, enroll, and manage wishlist.

## Features
- Course browsing
- Enrollment system
- Wishlist management
- Search, Sort, Pagination
- CRUD operations

## Tech Used
- FastAPI
- Python
- Pydantic

## Run Project
uvicorn main:app --reload
